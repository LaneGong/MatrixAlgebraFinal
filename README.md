## 结论

1. 人口较多、老龄化程度高的区县疫情发展情况相对更严重一些。

2. 通过高斯混合模型实现对4月22日后三天每日新增阳性病例数的预测，分别为13078人、11303人、9630人。可信度达到68.49%。

3. 上海本轮疫情发展拐点出现在4月13日左右，当日新增感染人数上限约为2.77万人。
